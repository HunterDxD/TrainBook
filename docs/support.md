# Support — TrainBook

Danke, dass du TrainBook nutzt!

## Häufige Fragen

**Wo werden meine Daten gespeichert?**
Lokal auf deinem iPhone. Optional werden sie über deine private iCloud synchronisiert — mehr dazu in der [Datenschutzerklärung](datenschutz.md).

**Wie erstelle ich ein Backup?**
In der App unter *Einstellungen → Backup* kannst du alle Daten exportieren und wieder importieren.

**Der Ruhe-Timer meldet sich nicht?**
Erlaube TrainBook Mitteilungen unter *iOS-Einstellungen → Mitteilungen → TrainBook*.
